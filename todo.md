## 1° passo

*Main menu*

- Criar um background infinito que se movimenta para direita. - OK
- Criar um bloco flutuante em cima do logo que irá ficar girando; - OK 
- Criar um sub menu que irá ficar aumentando e diminuindo; - OK
- Criar uma lista de 5 botoes largos e mais 2 botoes finos; - OK
  - Todos os botões terão um hover; - OK

## 2º passo

*Single player*

- Ao clicar no single player do menu.
  - O main menu irá ter display none e um display block no single player.
  - Irá abrir uma outra tela.
    - Esta tela terá um loading todo preto por 1 segundo.
      - Depois um Generating level Raising e uma barra de loading por .5s;
      - Depois um Eroding Raising e uma barra de loading por .5s;
      - Depois um Soiling level Raising e uma barra de loading por .5s;
      - Depois um Carving level Raising e uma barra de loading por .5s;
      - Depois um Growing level Raising e uma barra de loading por .5s;
    - E quando todo loading carregar irá abrir uma tela de invit/init.
      - Terá um h1 e 3 p;
      - 2 Label;
      - 2 Buttons;
        - Todos os botões terão um hover;
      - 1 Background estatico;
      - 1 Lista de ferramentas com hover crescente;
      - E uma funcinabilidade que irá dar display none na pagina de Single Player e um display block no main menu.
